## Flake8-f-string

flake8-f-string is a flake8 plugin to warn formatting string `f''`.

### Getting started
    
    $ pip install flake8-f-string
   

### Requirements

- Python 3
- flake8 >= 3.5

Error codes
-----------

This plugin is using the following error codes:

- FS100 - passing callable objects as arguments for formatted string
